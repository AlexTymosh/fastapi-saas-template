# Fastapi-SaaS-Template Backend

Backend package for the FastAPI SaaS template.

## Tooling and configuration

- Project configuration is managed through `pyproject.toml`.
- Development dependencies are defined as optional dependencies in
  `[project.optional-dependencies].dev`.
- Linting and formatting are handled by Ruff (`ruff check`) and ruff-format
  (`ruff format`).

## API documentation

- OpenAPI schema: `/openapi.json`
- Scalar UI: `/scalar`

For project overview, architecture, and setup instructions, see the repository root README.
